1) Что такое ветка в GIT

Ветка это указатель на коммит, который перемещается при создании нового коммита. При создании новой ветки и затем создания нового коммита в ней, переместится только указатель этой новой ветки. А указатель старой ветки останется на предыдущем коммите. Если переключится обратно на старую ветку и создать новый коммит, то получится разветвления. Как бы две новыч версии(дочерные) предыдушего коммита на каждый из которых указывает свой указатель ветки. В дальнейшем так же можно выполнить слияние этих веток.

2) Какое отношение имеют ssh-ключи к GITHUB

Если сохранить свой публичный ключ на GITHUB то локальный GIT репозиторий может взаимодействовать с GITHUB репозиторием без аутентификации

4) Укажите команды, которые выполняют след. действия (GIT):

создание, удаление веток

создать - git branch имя_ветки

удалить - git branch -d имя_ветки

переключение между ветками

git checkout имя_ветки

или

git switch имя_ветки

загрузка и удаление веток на GitHub

загрузка - git push -u origin имя_ветки

удаление - git push origin -d имя_ветки
